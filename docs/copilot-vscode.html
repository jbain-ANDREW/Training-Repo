What Is GitHub Copilot?

GitHub Copilot is an AI tool built into VS Code that suggests code as you type — similar to autocomplete, but for entire functions, blocks, and files. It also has a chat panel where you can ask questions, explain code, or request changes.

Copilot is made by GitHub (owned by Microsoft) and is powered by OpenAI models. It is deeply integrated into VS Code and works across all languages — Python, JavaScript, Google Apps Script, HTML, and more.

Copilot vs. Claude — Which Should You Use?

They are different tools that complement each other. Many developers use both.

Copilot is better for

  • Inline autocomplete while typing
  • Completing repetitive code patterns fast
  • Staying in flow without switching windows
  • Generating boilerplate quickly
  • Cost (free tier is generous)

Claude is better for

  • Longer back-and-forth conversations
  • Explaining complex logic in detail
  • Reasoning through architecture decisions
  • Debugging with full context
  • Writing from a detailed description
A practical workflow: use Copilot for the fast stuff while you type, and open Claude when you hit a problem that needs more thought or context.

Set Up Copilot in VS Code

  1. Create or sign in to your GitHub account

    Go to github.com. You already have one if you did the GitHub exercise.

  2. Enable Copilot on your account

    Go to github.com/settings/copilot and activate the free tier — or apply for the Student Developer Pack first if you qualify.

    Screenshot: GitHub Copilot settings page — Enable button [ screenshot: github.com/settings/copilot — Enable Copilot button ]
  3. Install the GitHub Copilot extension in VS Code

    Open the Extensions panel (Ctrl+Shift+X), search GitHub Copilot, and install the extension by GitHub. This also installs the Copilot Chat extension automatically.

    Screenshot: VS Code Extensions — GitHub Copilot by GitHub [ screenshot: VS Code Extensions — GitHub Copilot — Install button ]
  4. Sign in to GitHub from VS Code

    VS Code will prompt you to sign in to GitHub. Follow the browser prompt — it will redirect back to VS Code when done.

    Screenshot: VS Code GitHub sign-in prompt [ screenshot: VS Code — "Sign in to GitHub" notification ]
  5. Verify it's working

    Open any .py, .js, or .html file. Start typing and you should see grey suggestion text appear. Press Tab to accept a suggestion.

    Screenshot: Copilot inline suggestion appearing in VS Code [ screenshot: VS Code editor — grey autocomplete suggestion from Copilot ]

Using Copilot — Key Interactions

Inline autocomplete

Start typing and Copilot suggests code in grey. Press Tab to accept, Esc to dismiss, or keep typing to ignore it.

Write a comment, get the code

One of the most powerful tricks: write a comment describing what you want, press Enter, and Copilot will write the code beneath it.

Type this comment in a .py file, then press Enter # Read a CSV file and return a list of rows where the "score" column is above 80

Copilot Chat

Open the chat panel with Ctrl+Alt+I or click the Copilot icon in the sidebar. Use it exactly like Claude — paste code, ask questions, request changes.

Example chat prompt Here is my HTML splash page. Add a button that toggles between a light and dark background when clicked. Keep the existing styles. [paste your HTML]

Explain and fix with right-click

Select any code in the editor, right-click, and choose Copilot → Explain This or Fix This. Useful when you're reading unfamiliar code or hit an error.

Copilot learns from the code already in your file. The more context you give it — good variable names, clear comments, existing functions — the better its suggestions will be.
