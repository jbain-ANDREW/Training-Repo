What Is This?

This site is the starting point for your first GitHub exercise. You will set up VS Code, clone a repository, make a change, and submit it as a Pull Request. When your PR is merged, your work will appear on the Splash Page Archive.

Step 1 — Install the Tools

  • Install Git Download from git-scm.com/downloads and run the installer (defaults are fine).
  • Install VS Code Download from code.visualstudio.com and install.
  • Create a GitHub account Go to github.com and sign up if you don't have one.
  • Install the GitHub Pull Requests extension in VS Code Open VS Code → click the Extensions icon (left sidebar) → search GitHub Pull Requests → Install.

Step 2 — Clone the Repository

Open the integrated terminal in VS Code with Ctrl+` and run:

git clone https://github.com/jbain-ANDREW/Training-Repo.git
cd Training-Repo

VS Code will offer to open the folder — click Open.

Step 3 — Create Your Branch

Never work directly on main. Create your own branch first:

git checkout -b firstname-lastname

Replace firstname-lastname with your actual name, e.g. jane-smith.

Step 4 — Make Your Splash Page

Copy the template and rename it to match your branch:

cp exercises/splash-page/template.html docs/splash-pages/firstname-lastname.html

Open the new file in VS Code and customize it. See the template for guidance — change the title, colors, and content to make it yours.

Stuck? Open example.html to see a finished splash page.

Step 5 — Commit and Push

git add docs/splash-pages/firstname-lastname.html
git commit -m "Add splash page: Firstname Lastname"
git push origin firstname-lastname

Step 6 — Open a Pull Request

After pushing, GitHub will show a banner with a link to create a Pull Request. Click it, fill in the form, and submit. Your instructor will review and merge it.
