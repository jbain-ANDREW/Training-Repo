GitHub Exercise — Terminal

Clone, branch, commit, and open a Pull Request using terminal commands

Before You Start

This exercise uses the terminal. If you have never used one before, read the Terminal Basics guide first — it covers everything you need to follow along here.

You will also need a GitHub account and VS Code installed.

Prefer a visual app instead of typing commands? Do this exercise with GitHub Desktop — or do both and compare.

Step 1 — Install the Tools

  • Install Git

    Download from git-scm.com/downloads and run the installer. The defaults are fine.

  • Install VS Code

    Download from code.visualstudio.com and install.

  • Install the GitHub Pull Requests extension in VS Code

    Open VS Code → Extensions panel (Ctrl+Shift+X) → search GitHub Pull Requests → Install.

Step 2 — Clone the Repository

Open the integrated terminal in VS Code with Ctrl+` and run:

git clone https://github.com/jbain-ANDREW/Training-Repo.git
cd Training-Repo

VS Code will offer to open the folder — click Open.

What just happened? Cloning copies the entire repository from GitHub to your machine. You now have a local copy you can edit.

Step 3 — Create Your Branch

Never work directly on main. Create your own branch first:

git checkout -b firstname-lastname

Replace firstname-lastname with your actual name in lowercase, e.g. jane-smith.

What is a branch? A branch is your own private copy of the project where you can make changes without affecting anyone else's work. When your changes are ready, you merge them back via a Pull Request.

Step 4 — Make Your Splash Page

Copy the template and rename it to match your name:

cp exercises/splash-page/template.html docs/splash-pages/firstname-lastname.html

Open the new file in VS Code and customize it. The template has numbered comments guiding you through each change — title, background color, emoji, and personal info.

Stuck on what to write? Open example.html to see a finished page.

Step 5 — Commit Your Changes

git add docs/splash-pages/firstname-lastname.html
git commit -m "Add splash page: Firstname Lastname"
What is a commit? A commit is a saved snapshot of your changes with a message explaining what you did. Think of it as a checkpoint you can always return to.

Step 6 — Push and Open a Pull Request

git push origin firstname-lastname

After pushing, go to github.com/jbain-ANDREW/Training-Repo. GitHub will show a banner: "firstname-lastname had recent pushes — Compare & pull request." Click it, fill in the form, and submit.

Once your instructor reviews and merges it, your page will appear in the Splash Page Archive.

Only edit files inside docs/splash-pages/. Changes outside that folder will not be accepted.
