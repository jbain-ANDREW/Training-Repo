Before You Start

This guide assumes you have never installed Python before. If you already have some version of Python on your machine, use the Already Installed guide instead.

You will need a Windows machine with internet access and administrator rights (most personal laptops qualify).

Step 1 — Download Python

  1. Go to python.org/downloads

    Open a browser and navigate to python.org/downloads. The site will detect that you are on Windows and show a large download button for the latest stable release.

    Screenshot: python.org/downloads — Windows download button [ screenshot: python.org/downloads — the yellow "Download Python 3.x.x" button ]
  2. Download the installer

    Click the button. Your browser will download a file named something like python-3.12.x-amd64.exe. Save it somewhere easy to find (Desktop is fine).

    Do not install Python from the Microsoft Store. It looks similar but causes PATH problems. Always use the installer from python.org.

Step 2 — Run the Installer

  1. Launch the installer

    Double-click the downloaded .exe file. Windows may ask for permission — click Yes.

  2. Check "Add python.exe to PATH" — this is critical

    On the first screen of the installer you will see a checkbox at the bottom. Check it before clicking anything else. Without this, Python won't be accessible from the terminal.

    Screenshot: Installer first screen — "Add python.exe to PATH" checkbox highlighted [ screenshot: Python installer — checkbox at bottom reads "Add python.exe to PATH" — must be checked ]
  3. Click "Install Now"

    Use the default "Install Now" option (not "Customize installation"). The defaults are correct for most students.

    Screenshot: Installer progress bar [ screenshot: installer progress bar ]
  4. Click "Close" when finished

    When you see "Setup was successful", the installation is complete.

    Screenshot: "Setup was successful" screen [ screenshot: green checkmark — "Setup was successful" ]

Step 3 — Verify the Installation

  1. Open a terminal

    Press Win + R, type cmd, press Enter. Or open VS Code and press Ctrl+` to open the integrated terminal.

    Screenshot: Windows Run dialog with "cmd" typed [ screenshot: Win+R dialog with "cmd" ]
  2. Check the Python version

    Type the following and press Enter:

    python --version

    You should see something like Python 3.12.3. If you see an error, see the troubleshooting note below.

    Screenshot: Terminal showing "Python 3.12.x" output [ screenshot: cmd window — python --version → Python 3.12.x ]
  3. Check pip is available

    pip is the tool for installing Python packages. Verify it is present:

    pip --version

    You should see a version number and a file path. If you do, you are all set.

Getting "python is not recognized"? The PATH checkbox was probably missed. The easiest fix is to re-run the installer, choose "Modify", and look for the option to add Python to PATH. Or see the Already Installed guide for how to fix PATH manually.

Step 4 — Install the VS Code Python Extension

  1. Open VS Code

    If you haven't installed VS Code yet, do that first — see the main guide.

  2. Open the Extensions panel

    Click the Extensions icon in the left sidebar (looks like four squares), or press Ctrl+Shift+X.

    Screenshot: VS Code Extensions panel open [ screenshot: VS Code — Extensions panel open ]
  3. Search for "Python" and install

    Type Python in the search box. The first result should be the official extension by Microsoft. Click Install.

    Screenshot: VS Code — Python extension by Microsoft in search results [ screenshot: VS Code Extensions — "Python" by Microsoft — Install button ]

Step 5 — Run Your First Script

  1. Create a new file in VS Code

    Press Ctrl+N to open a new file. Save it as hello.py (Ctrl+S).

  2. Type a simple program 
    print("Hello, world!")
  3. Run it

    Press Ctrl+F5 or click the Run button (▷) in the top-right corner. You should see Hello, world! appear in the terminal panel at the bottom.

    Screenshot: VS Code — terminal showing "Hello, world!" output [ screenshot: VS Code terminal — Hello, world! ]
If VS Code asks you to select a Python interpreter, choose the one that shows the version you just installed (e.g., Python 3.12.x).
