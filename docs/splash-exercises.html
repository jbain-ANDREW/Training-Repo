Before you start: you need a splash page from the GitHub exercise. Open that file in VS Code, then open Claude or Copilot Chat alongside it. For each exercise below, paste your current HTML into the chat along with the prompt.

Exercise 1 — Animated Entrance

Add CSS animations so your name and content fade and slide in when the page loads. Pure CSS — no JavaScript needed.

1
Fade and slide everything in on load
Beginner

See what this looks like before you build it:

▶ View live example

The example uses CSS @keyframes with staggered animation-delay values so each element appears slightly after the one above it. The background also slowly shifts through a gradient loop.

Prompt to use with Claude or Copilot

Claude / Copilot Chat
Here is my HTML splash page: [paste your full HTML here] Please add CSS animations so that: - The emoji avatar fades in and drops down from above - The name slides up and fades in 0.3 seconds later - The tagline slides up 0.5 seconds after that - The info box slides up 0.7 seconds after that Also add a slowly shifting gradient animation to the page background. Use only CSS — no JavaScript. Keep all my existing styles and content.
What to look for in the response Claude should add @keyframes rules and animation properties with different delays. If the animations feel too fast or slow, follow up: "Make the animations slower" or "Increase the delay between elements."

Exercise 2 — Dark / Light Toggle

Add a button that switches the page between dark and light mode. Introduces CSS variables and a tiny bit of JavaScript.

2
Toggle dark and light mode with a button
Beginner
▶ View live example (includes skill tags)

The example uses CSS custom properties (--bg, --text, etc.) so that toggling a single class on <body> switches all the colors at once. The JavaScript is one line: document.body.classList.toggle('light').

Prompt to use with Claude or Copilot

Claude / Copilot Chat
Here is my HTML splash page: [paste your full HTML here] Add a dark/light mode toggle button. Requirements: - The button should appear in the top-right corner of the page - Clicking it switches between the current dark theme and a clean light theme - Use CSS custom properties (variables) for all colors so the switch is smooth - Add a CSS transition so the background and text color fade over 0.3 seconds - Use the minimum JavaScript needed — ideally just one line - Label the button with ☀️ / 🌙 Keep all my existing content.
What to look for in the response Look for :root with CSS variables and a body.light override block. If the button is in the wrong position, ask: "Move the button to the top-right corner and keep it there when scrolling."

Exercise 3 — Skills Tags

Add a row of color-coded pill badges showing your skills, tools, or interests.

3
Add colored skill / interest tags
Intermediate
▶ View live example

Each tag is a <span> with a class. Different classes get different background colors. Tags for things you're learning get a different style (outlined instead of filled).

Prompt to use with Claude or Copilot

Claude / Copilot Chat
Here is my HTML splash page: [paste your full HTML here] Add a row of skill/interest tags below my name. Requirements: - Tags should be pill-shaped (rounded) with colored backgrounds - Include these skills: Python, JavaScript, HTML/CSS, Google Apps Script - Include these "currently learning" items: [add your own] — style these differently (outline only, no fill) - The tags should wrap on smaller screens - Make it look clean and modern Place the tags between my name/tagline and the info box.
What to look for in the response Look for display: flex; flex-wrap: wrap on the container. To change colors, follow up: "Make the Python tag blue and the JavaScript tag yellow."

Exercise 4 — Typewriter Effect

Make your tagline cycle through several phrases using a typewriter animation. Introduces basic JavaScript logic.

4
Cycle through phrases with a typing animation
Intermediate
▶ View live example

The JavaScript types one character at a time, pauses when the phrase is complete, then deletes it and moves to the next. The blinking cursor is pure CSS.

Prompt to use with Claude or Copilot

Claude / Copilot Chat
Here is my HTML splash page: [paste your full HTML here] Replace my static tagline with a typewriter effect that cycles through these phrases: - "Student. Builder. Tinkerer." - "I write Python and break things." - "Currently learning [your thing here]." - "I use AI to code faster." Requirements: - Type each phrase one character at a time - Pause for 1.5 seconds when a phrase is complete - Delete it character by character, then start the next phrase - Show a blinking cursor while typing (CSS only) - Loop forever Write this in plain JavaScript — no libraries.
What to look for in the response Look for a setTimeout or setInterval loop and an array of phrases. To adjust speed, ask: "Make the typing faster" or "Make the delete speed twice as fast as the typing speed."

Exercise 5 — Put It All Together

Combine everything into one polished page, then submit it as a Pull Request to replace your original splash page.

5
Design your definitive splash page
Challenge

Take the best pieces from the previous exercises and ask Claude or Copilot to combine them into one cohesive page. Use this as a chance to make it genuinely yours — your name, your colors, your interests.

Prompt to use with Claude or Copilot

Claude / Copilot Chat
I want to build a polished personal splash page. Here are the features I want: 1. Animated entrance — elements fade/slide in on load with staggered delays 2. Dark/light toggle button in the top-right corner 3. Color-coded skill tags: [your skills] 4. Typewriter effect on my tagline cycling through: [your phrases] 5. My name: [your name] 6. My color scheme: [describe it, e.g. "dark navy and white" or "warm orange tones"] Write the complete HTML file — everything in one file, no external dependencies. Make it look modern and clean.
After you have a page you like Save it as docs/splash-pages/firstname-lastname.html in the repo, commit it on your branch, and open a Pull Request. It will appear in the Splash Page Archive.
