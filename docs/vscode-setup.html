What Is VS Code?

Visual Studio Code (VS Code) is a free code editor made by Microsoft. It works for every language covered in this guide — Python, JavaScript, HTML, CSS, Google Apps Script — and it's the environment where AI tools like Claude and GitHub Copilot plug in directly.

Think of VS Code as the hub: your code lives here, your terminal is here, Git is here, and your AI assistant is here. Learning one editor well is more valuable than switching between many.

VS Code is not just for Python. The same editor you use to run a Python script works equally well for editing HTML, writing JavaScript, or reviewing a Google Apps Script file that Claude generated for you.

Install VS Code

  1. Download the installer

    Go to code.visualstudio.com and click the big blue Download for Windows button. This gives you the User Installer — no admin rights needed.

    Screenshot placeholder code.visualstudio.com — Download for Windows button
    save as: assets/images/vscode-download.png
  2. Run the installer

    Open the downloaded .exe and accept the defaults. On the "Select Additional Tasks" screen, check Add to PATH and Open with Code (both right-click options). These make VS Code easy to launch from the terminal.

    Screenshot placeholder Installer — "Add to PATH" and "Open with Code" checkboxes
    save as: assets/images/vscode-installer-options.png
  3. Open VS Code

    Launch from the Start menu or by typing code in any terminal window. The first launch shows a Welcome tab — you can close it.

Getting Oriented

VS Code has five main areas. You'll use all of them within the first few minutes.

Explorer (left sidebar)

Your file tree. Open a folder with File → Open Folder and all files appear here. Click any file to open it.

Editor (center)

Where you write code. Multiple files open as tabs. Split the editor with View → Editor Layout to see two files side by side.

Terminal (bottom)

A full terminal inside VS Code. Open with Ctrl+` (backtick) or Terminal → New Terminal. Runs in the folder you have open.

Source Control (left sidebar)

Git integration. Shows changed files, lets you stage, commit, and push without leaving the editor. The branch name shows in the bottom-left corner.

Extensions (left sidebar)

The marketplace for add-ons. Search by name and click Install. Extensions add language support, AI tools, themes, and more.

Command Palette

Ctrl+Shift+P opens a searchable list of every VS Code action. If you can't find something in a menu, type it here.

Recommended Extensions

Install extensions from the Extensions panel (Ctrl+Shift+X). Search the name, click the result, click Install.

Essential
Python
Microsoft

Language support for Python — syntax highlighting, linting, debugging, and running scripts. Required if you write any Python.

Essential
Pylance
Microsoft

Fast IntelliSense and type checking for Python. Usually installed automatically with the Python extension.

AI
GitHub Copilot
GitHub

Inline code suggestions as you type, plus a chat panel. Free tier available; free for verified students. See the Copilot page for setup.

AI
Claude for VS Code
Anthropic

Claude AI directly in your editor. Ask questions, generate code, explain errors. See the Claude page for setup and prompting tips.

Essential
GitLens
GitKraken

Enhances the built-in Git support — shows who changed each line, full commit history, branch comparisons, and more.

Optional
Prettier
Prettier

Auto-formats HTML, CSS, and JavaScript on save. Keeps code consistently readable without manual cleanup.

Don't install too many extensions at once. Start with Python and one AI tool. Add others as you hit a need for them — a cluttered editor slows you down.

Opening a Project

VS Code works best when you open an entire folder, not just individual files. This gives it the full context of your project — file tree, Git history, terminal path.

  1. File → Open Folder

    Navigate to your project folder (e.g., your cloned Training-Repo) and click Select Folder. VS Code reloads with that folder as the root.

  2. Trust the folder

    VS Code may ask "Do you trust the authors of the files in this folder?" — click Yes, I trust the authors for folders you created or cloned yourself.

  3. Open the terminal

    Press Ctrl+`. The terminal opens at the root of your project folder — you're ready to run Python scripts, Git commands, or anything else.
