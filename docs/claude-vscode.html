What Is Claude?

Claude is an AI assistant made by Anthropic. In a coding context it can write code from a description, explain what existing code does, find bugs, suggest improvements, and help you think through a problem. It works across languages — Python, JavaScript, Google Apps Script, HTML/CSS, and more.

Unlike a search engine, Claude understands context. You can paste in your code, describe what's wrong, and get a specific answer — not a list of links.

There is no dedicated student discount for Claude as of early 2026. If budget is a concern, start free and compare with GitHub Copilot, which is free for verified students.

What Claude Can and Can't Do

Claude is great at

  • Writing code from a description ("write a Python function that...")
  • Explaining code you paste in ("what does this do?")
  • Debugging ("here's my error — what's wrong?")
  • Refactoring and improving existing code
  • Generating Google Apps Script, JavaScript, HTML/CSS
  • Talking through approach before you write anything

Claude is not good at

  • Real-time information (knowledge has a cutoff date)
  • Running your code — it can write it, but you have to run it
  • Knowing what's actually on your machine (unless you tell it)
  • Always being right — verify its output, especially for logic-heavy code

Set Up Claude in VS Code

This requires a Claude Pro account ($20/month). If you are on the free tier, you can still use claude.ai in your browser alongside VS Code — just paste code back and forth.
  1. Create a Claude account

    Go to claude.ai and sign up. Free tier is fine to start.

  2. Open the Extensions panel in VS Code

    Press Ctrl+Shift+X and search for Claude. Install the extension by Anthropic.

    Screenshot: VS Code Extensions — Claude extension by Anthropic [ screenshot: VS Code Extensions panel — Claude by Anthropic — Install button ]
  3. Sign in

    After installing, a Claude icon will appear in the left sidebar. Click it and sign in with your Anthropic account.

    Screenshot: Claude sidebar open in VS Code [ screenshot: VS Code — Claude icon in sidebar — Sign in prompt ]
  4. Open a chat

    Click the Claude icon to open the chat panel. You can now ask questions, paste code, and get responses without leaving VS Code.

How to Talk to Claude (Prompting Basics)

The quality of Claude's response depends heavily on how clearly you describe what you want. A few patterns that work well:

Give context, then ask

Example prompt I have a Python script that reads a CSV file and calculates the average of a column called "score". It works but crashes when the column has empty cells. Here's the code: [paste code] Fix the crash and explain what was wrong. Tip: paste your actual code where it says [paste code]

Describe the goal, not just the symptom

Less useful Why doesn't this work?
More useful I'm trying to make a button on my HTML page change the background color when clicked. I wrote this JavaScript but nothing happens when I click. Here's my code: [paste code] What am I missing?

Ask for options, not just an answer

Example prompt I want to store a small list of names and scores in my Google Apps Script. What are two or three ways to do that, and what are the tradeoffs?
If Claude's first answer isn't quite right, don't start over — just reply and refine. "That almost works but the animation is too fast" or "Can you do that without jQuery?" are perfectly good follow-ups.
